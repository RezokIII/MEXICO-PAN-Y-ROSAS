# ART_TODO — image shopping list
## La Izquierda Mexicana / México: Pan y Rosas

**How to use this list:** find or make the image, name the file EXACTLY as shown,
drop it in `out/html/img/` (portraits in `out/html/img/portraits/`, card art in
`out/html/img/cards/`). Tell Claude which files landed and the one-line scene
references get wired. Nothing renders broken in the meantime.

**Style guide:** high-contrast two-color silkscreen / gráfica popular of '68 looks
best and avoids rights problems. Photos: public domain / Wikimedia Commons, log
sources in credits_images.txt. Keep files under ~200KB. Cards ~350px wide,
face-images ~700px, portraits ~256px square-ish.
